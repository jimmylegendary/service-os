# PORTFOLIO.md

## Active services

### 1. Today It
Status: primary focus

A today-first action system.
Not a traditional scheduler or backlog-heavy todo manager.
Its purpose is to compress mental clutter into a small, actionable present surface.

### 2. Study It
Status: secondary / dependent on Today It foundations

A purpose-driven study and goal-oriented group coordination app.
More operational than social.
Likely to inherit task and flow concepts from Today It.

### 3. AI workflow as API service
Status: exploratory / platform layer

A platform for exposing AI workflows as reusable services so creators can monetize them and users can consume narrow-fit AI workflows.
Should likely be approached after internal workflow execution patterns are understood in practice.

## Longer-term directions

- progressive automation of product creation and operation
- MCP/API-mediated human-AI task exchange
- reusable product-building and deployment protocols
- service factory / operating model for multi-product work

1. establish central design and working system

4. create Today It implementation repo
